# 🔩 Bearing Resistance Prediction using Machine Learning

A Machine Learning web application that predicts **bearing resistance** using multiple regression algorithms. The project includes data preprocessing, model training, evaluation, and a deployed web interface for real-time predictions.

## 🚀 Live Demo

🌐 **Website:** https://bearing-resistance-ml.vercel.app/

## 📂 GitHub Repository

https://github.com/isshhiittaaa/Bearing-Resistance-Prediction

## ✨ Features

- Predicts bearing resistance from input parameters
- Data preprocessing and feature scaling
- Multiple regression algorithms
- Model comparison
- Interactive web interface
- Real-time prediction
